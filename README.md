# Transaction Reconciliation Engine

A Node.js backend application built with Express and MongoDB to reconcile financial transaction data from CSV files.

## Tech Stack
- Node.js
- Express
- MongoDB with Mongoose
- csv-parser for ingestion

## Setup
1. Clone the repository
2. Run `npm install`
3. Copy `.env.example` to `.env` and fill in DB connection uri
4. Start server: `node index.js`
